ENABLE_VNC=1 \

MT5_INSTALL_MODE=manual \
ENABLE_VNC=1 \
docker compose up mt5-server


=================
docker compose build --no-cache
-> usar o no-cache apenas se tentativas anteriores foram muito ruins então baixa tudo novo. mas isso deixa bem demorado

docker compose up
-> inicia no docker

docker compose logs -f
-> para conferir log internos

docker compose down -v
-> Derruba o ambiente e deleta o volume persistente (usar caso tenha que alterar algo no processo de criacao da imagem)
